===================================

   - Ingresar usuario y contraseña
   
2. CREAR USUARIO (Solo Admin)
   - Dashboard > Crear Usuario
   - Completar formulario
   - Guardar
   
3. CREAR CONTRATO
   - Dashboard > Nuevo Contrato
   - Completar todas las secciones
   - Los campos se muestran/ocultan según selección
   - Cargar documentos
   - Guardar
   
4. VER CONTRATOS
   - Dashboard > Listar Contratos
   - Usar filtros para buscar
   - Click en "Ver" para detalles
   - Click en "Documentos" para ver archivos

===================================
CAMPOS DINÁMICOS
===================================

El formulario muestra/oculta campos según:

- Si tiene DISCAPACIDAD = Sí
  → Muestra: "Especifique cuál"
  
- Si tiene HIJOS MENORES = Sí
  → Muestra: "¿Cuántos?"
  
- NIVEL DE ESTUDIO:
  - Técnico → Muestra: Formación técnica
  - Tecnólogo → Muestra: Formación tecnológica
  - Profesional → Muestra: Formación pregrado
  - Posgrado → Muestra: Opciones de posgrado
  
- DATOS DE POSGRADO:
  - Maestría → Muestra: Campo maestría
  - Doctorado → Muestra: ¿Posee doctorado?

===================================
SOLUCIÓN DE PROBLEMAS COMUNES
===================================

PROBLEMA: "Error de conexión a la base de datos"
SOLUCIÓN: 
- Verificar que XAMPP esté ejecutándose
- Abrir phpMyAdmin y verificar que existe la BD "contratos"

PROBLEMA: "No se pueden subir archivos"
SOLUCIÓN:
- Dar permisos a la carpeta "uploads"
- Verificar que el archivo no exceda 5MB

PROBLEMA: "Sesión expirada constantemente"
SOLUCIÓN:
- En php.ini aumentar session.gc_maxlifetime = 1440

PROBLEMA: "Usuario o contraseña incorrectos"
SOLUCIÓN:
- Verificar credenciales (admin / admin123)
- Importar nuevamente database.sql
